Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents, typically HTML or XML, that defines the structure, style, and content of a webpage. It represents the document as a tree-like structure where every part of the document (elements, attributes, text, etc.) is treated as an object that can be accessed and manipulated by programming languages like JavaScript.

How do you select an element from the DOM?

getElementById: Selects an element by its ID.
Example: const element = document.getElementById('elementId');
getElementsByClassName: Selects all elements with a specific class name (returns an HTMLCollection).
Example: const elements = document.getElementsByClassName('className');
getElementsByTagName: Selects all elements with a specific tag name (returns an HTMLCollection).
Example: const elements = document.getElementsByTagName('tagName');
querySelector: Selects the first element that matches a CSS selector.
Example: const element = document.querySelector('.className'); (by class)
querySelectorAll: Selects all elements that match a CSS selector (returns a NodeList).
Example: const elements = document.querySelectorAll('.className'); (by class)

What is event delegation in the context of the DOM and Why is it useful?

Event delegation is a technique in JavaScript where a single event listener is added to a parent element instead of adding individual event listeners to multiple child elements. This is done by taking advantage of event bubbling, where events propagate up the DOM tree from the target element to its ancestors.

How do you manipulate an element's attribute and styles using the DOM?

To manipulate an element's attributes and styles using the DOM in JavaScript, you can use methods like getAttribute, setAttribute, and removeAttribute to manage attributes. For direct access, you can modify common attributes as properties (e.g., element.id). To change styles, use the style property to set inline styles, such as element.style.color = 'red'. You can also set multiple styles at once using Object.assign. To remove a style, simply set the property to an empty string (e.g., element.style.color = '').
